# 侧边栏同步状态增强

## 用户故事

作为用户，我希望侧边栏底部能一眼看到设备连接和同步状态，以便随时了解笔记是否已同步。

## 依赖

- 无依赖（L0，可独立开始）
- v0.2.0 侧边栏底部已有网络状态指示，本功能在此基础上增强

## 需求描述

当前侧边栏底部显示简单的网络状态指示（状态点 + 「已连接」/「未连接」文字 + 连接设备数）。v0.2.1 将其增强为同步状态视角，让用户更直观地了解同步情况。

## 交互设计

### 状态展示

根据不同场景显示不同信息：

**已连接 + 已同步**：
```text
┌────────────────────────────────┐
│  🟢 2 台设备在线 · 已同步      │
└────────────────────────────────┘
```

**已连接 + 同步中**：
```text
┌────────────────────────────────┐
│  🟢 2 台 · 同步中 40/100       │
└────────────────────────────────┘
```

**未连接**：
```text
┌────────────────────────────────┐
│  ⚪ 未连接                      │
└────────────────────────────────┘
```

**网络未启动**：
```text
┌────────────────────────────────┐
│  ⚪ 网络未启动                   │
└────────────────────────────────┘
```

### 交互行为

- 点击整行 → 打开设置窗口的「同步」Tab（`/settings/sync`）
- hover 显示 tooltip：如「已同步到 MacBook Pro、Desktop-Win」或「点击查看同步详情」

### 与现有设计的关系

替换当前侧边栏底部的网络状态行。[sidebar-settings-layout.md](../../v0.2.0/features/sidebar-settings-layout.md) 中规划了侧边栏底部精简为单行网络状态，本功能在此基础上将内容从「网络状态」升级为「同步状态」。

## 技术方案

### 前端

**改造 Sidebar 底部组件**：

当前 `Sidebar.tsx` 底部有一个网络状态按钮，需要改造为同步状态展示：

```typescript
// 状态优先级（从高到低）：
// 1. 网络未启动 → "网络未启动"
// 2. 未连接任何设备 → "未连接"
// 3. 全量同步中 → "N 台 · 同步中 X/Y"
// 4. 已连接已同步 → "N 台设备在线 · 已同步"
```

**数据来源**：

- 设备连接数：`useNetworkStore` 的 `connectedPeers`
- 同步状态/进度：监听 `sync-progress` 事件，或从 `useSyncStore` 读取（设置页同步 Tab 共用）

**组件**：

```text
src/components/layout/Sidebar.tsx    # 改造底部区域
src/components/layout/SyncStatus.tsx # 新增：同步状态行组件（可复用）
```

### 后端

- 无新增后端功能，复用已有的 Tauri 事件和 commands
